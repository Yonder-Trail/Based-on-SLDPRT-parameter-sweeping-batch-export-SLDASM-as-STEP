# Based-on-SLDPRT-parameter-sweeping-batch-export-SLDASM-as-STEP
本项目依托 SolidWorks COM API 与 Python 实现自动化建模功能，可通过修改零件全局变量驱动装配体参数化重建，并完成 STEP 文件的批量导出。  Leverages SolidWorks COM API and Python for automated modeling. Enables assembly parametric reconstruction through modification of part global variables, with full support for batch STEP file export.
